# Meter Tracker

Just a fun little app I wanted to build for my current job.

## What's Inside

- [Bun](https://bun.sh/) - A fast all-in-one JavaScript runtime.
- [Next.js](https://nextjs.org/) - The React framework for production.
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapidly building custom user interfaces.
- [Shadcn UI](https://ui.shadcn.com/) - A component library for building beautiful and accessible user interfaces.
- [Biome](https://biomejs.dev/) - A fast and powerful code formatter and linter.

## Getting Started

First, run the development server:

```bash
bun -b run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Production

```bash
bun -b run build
```
